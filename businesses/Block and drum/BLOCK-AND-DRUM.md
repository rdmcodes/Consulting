## Website Build — Vision & Vibe Direction

### The Feel
- **Nighttime mood:** Dark, moody, gold accents — think 90s R&B vibes, laid back, head-bobbing energy
- **Color palette:** Deep blacks, dark golds, warm amber tones — luxe but not pretentious
- **Daytime contrast:** They serve coffee in the morning (Cafe Studio) — the site needs to acknowledge both sides: cozy morning cafe AND nightlife venue
- **Overall energy:** You NEED to be in the loop. There's always something happening — date nights, live R&B, outdoor concerts, games, food trucks, vinyl nights. The site should make you feel like you're missing out if you're not checking in

### What the Site Should Communicate
- This place provides SO MUCH — distillery, live music, vinyl lounge, cafe, food trucks, courtyard events, games, date nights
- You need to stay updated to know what vibe it's going to be on any given night — who's performing, what's the theme
- The variety IS the selling point — every visit is different

### Design Direction
- Sections flow together — not too far apart, feels like one cohesive scroll but with distinct sub-sections
- Gallery with real photos and correct captions underneath
- Split layouts / sectioned content when scrolling — but tight, not spread out
- Heavy use of motion, animation, scroll transitions — leverage 21st.dev MCP for design patterns
- Pictures and movement should carry the experience — the site should FEEL like being there
- Inspiration sources: 21st.dev component library, additional links TBD from Matthew

### Sections Likely Needed
1. Hero — dark, moody, animated, sets the tone immediately
2. What's Happening / Events — the heartbeat of the site, always current
3. The Distillery — hand-pick botanicals, custom spirits, vacuum distillation
4. Vinyl Room — listening lounge, 100 seats, bring-your-own-vinyl nights
5. The Courtyard — 8,000 sq ft, live music stage, outdoor seating
6. Cafe Studio — morning coffee, pastries, Rail Trail patio (lighter aesthetic)
7. Food & Chefs — shipping container kitchens, rotating food trucks

## Design Inspiration Collected

### Adam Bricker (Awwwards SOTD)
- Fluid typography with `clamp()` functions — scales with viewport
- Inter Tight font family (weights 300-800)
- Color palette with CSS variables — orange #FA5D29, green #AAEEC4, purple #502BD8
- Dark mode toggle via `.dark` body class
- Sticky floating header that animates into view
- Animated gradient link underlines on hover
- Pulse animations for notification badges
- Marquee text animations
- Grid card systems with image overlays and layered structures
- 0.3-0.6s transitions with ease-out timing

### 21st.dev Components to Translate to Vanilla HTML
1. **Animated Scroll Gallery** — 3-column parallax gallery with scroll-driven 3D rotation (rotateX 75° → 0°), scale transforms, blur-in text animations. Translate to CSS scroll-driven animations + IntersectionObserver
2. **Immersive Scroll Gallery** — Images zoom outward on scroll (scale 1 → 4-9x), fade to reveal text content underneath. Could work for hero → content transition
3. **Scroll Trigger Animations** — Smooth scroll (Lenis), inset clip-path animations, scale/translate transforms tied to scroll position, rounded corner morphing

### Key Techniques to Use (Vanilla HTML + Tailwind CDN)
- IntersectionObserver for scroll-triggered reveals
- CSS `scroll-timeline` / `animation-timeline: scroll()` for scroll-linked animations
- CSS `clip-path: inset()` for image reveals
- `transform: scale()` + `translateY()` for parallax columns
- `filter: blur()` → `blur(0)` for text entrance
- CSS custom properties for the dark gold color palette
- `mix-blend-mode: screen` for ambient glow effects
- Smooth scroll via CSS `scroll-behavior: smooth` or lightweight JS
